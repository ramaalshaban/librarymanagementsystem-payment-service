Welcome to librarymanagementsystem-payment Service

Handles overdue fee assessment, payment resolution (member and staff), Stripe integration, fee/payment audit trail, and payment oversight for the Smart Library Management System.

Available Routes

  • getFee => GET /fees/:feeId
  • createFee => POST /fees
  • updateFee => PATCH /fees/:feeId
  • deleteFee => DELETE /fees/:feeId
  • listFees => GET /fees
  • checkoutstartFee => PATCH /startcheckout/fee/:feeId
  • checkoutcompleteFee => PATCH /completecheckout/fee/:feeId
  • checkoutrefreshFee => PATCH /refreshcheckout/fee/:feeId
  • getFeePayment => GET /feepayments/:feePaymentId
  • createFeePayment => POST /feepayments
  • updateFeePayment => PATCH /feepayments/:feePaymentId
  • deleteFeePayment => DELETE /feepayments/:feePaymentId
  • listFeePayments => GET /feepayments
  • getFeeEvent => GET /feeevents/:feeEventId
  • createFeeEvent => POST /feeevents
  • updateFeeEvent => PATCH /feeevents/:feeEventId
  • deleteFeeEvent => DELETE /feeevents/:feeEventId
  • listFeeEvents => GET /feeevents
  • getPayment => GET /payment/:feePaymentId
  • getPaymentByOrderId => GET /paymentbyorderid/:feePaymentId
  • getPaymentByPaymentId => GET /paymentbypaymentid/:feePaymentId
  • createPayment => POST /payment
  • updatePayment => PATCH /payment/:feePaymentId
  • listPayments => GET /payments
  • deletePayment => DELETE /payment/:feePaymentId
